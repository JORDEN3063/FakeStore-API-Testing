# FakeStore API Testing Project

## Overview
API testing of an e-commerce backend using Postman.

## APIs Covered
- User Create
- User Update
- User Delete
- Products GET
- Auth/Login
- Cart operations

## Tests Included
- Positive scenarios
- Negative scenarios
- Environment variables
- Collection Runner usage

## How to Run
1. Import collection in Postman
2. Set environment FakeStore
3. Run collection runner
